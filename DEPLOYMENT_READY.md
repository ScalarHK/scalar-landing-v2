## Common Questions

### Q: Do I need a backend?
A: **Not for this MVP.** The landing page is fully functional without it. Add backend later when you're ready to scrape real websites.

### Q: How much does hosting cost?
A: **Free.** Vercel free tier covers unlimited websites like this. Upgrade to paid only if you need very high traffic.

### Q: Can I use a custom domain?
A: **Yes, optionally.** Works with `yourdomain.com` instead of Vercel's subdomain. Domain costs $10-15/year.

### Q: What if I want to change colors/copy after launch?
A: **Super easy.** Edit files locally, push to GitHub, Vercel redeploys automatically. Takes 1-2 minutes.

### Q: Can I track user behavior?
A: **Yes, add Google Analytics.** Instructions in `VERCEL_DEPLOYMENT.md`.

### Q: What happens if my code breaks?
A: **Rollback in 1 click.** Vercel dashboard → Deployments → Select previous version → Promote to Production.
